# class_predicates_extractor

Java utility that computes class-predicate usage statistics from RDF triples for *PYTHIA*. Use this after `tools/elements_extractor`.

## Build

```bash
mvn clean package
```

## Usage Example

```bash
java -jar target/class-predicates-extractor-1.0-SNAPSHOT-jar-with-dependencies.jar \
  -i ~/freebase.nt \
  -cf ~/extracts_freebase/classes.tsv \
  -o ~/indices_freebase/classes_predicates.tsv \
  -t 16
```

## Arguments

- `-i, --input` (required): input RDF file or directory.
- `-cf, --class-file` (required): class URI list file. You can use the `classes.tsv` for the target knoweldge graph, which is generated by `tools/elements_exractor`.
- `-o, --output` (required): output TSV file.
- `-t, --threads`: worker threads (default: available processors).

## Important

Although you can choose the name of the output, two files will be generated. Those two files must have the exact names specified in the `README.md` file of the project root. Examples can be found in the `README.md` file in the `indices/` directory.
