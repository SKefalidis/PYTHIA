# LLM Baselines

In this folder you can find files for running our baseline models. To do so use the scripts that start with `generate_`.

After retrieving outputs you can calculate metrics as is done for *PYTHIA* and the rest of the engines (see relevant `README.md` files).

#### Note: By default, more experiments are setup than what fit in the paper. If your aim is to reproduce our results we recommend that you disable those.
